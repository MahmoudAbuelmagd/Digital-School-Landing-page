The learning path that grows with your child

Transition smoothly from visual building blocks to real-world text-based programming.

Blocks level 1–2

Start with “building blocks” and core logic. Perfect for beginners and younger learners.

Mix level 3

Combine blocks and text side-by-side to build confidence and language fluency.

Text level 4

Move into text-based programming and real tools used in game and app development.
